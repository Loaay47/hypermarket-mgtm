# Project 9: Hyper Market Management System

This system can be installed in Large Markets to Manage sales and purchase.
users of the system have different types with different roles.

This system contains the following modules:

---
## 1. Administrative Module

### admin data field
* username
* password
can alter them

### Admin manages employees in this way:
#### actions on employees ddl
* add
* set unique id
* set password
* set a role

#### admin actions dml
* new employee
* delete
* update
* list all employees
* search

`(Add - set unique id, password and type to - new
employee, Delete, update, List all Employees, search)`

---
## 2. Marketing Module

### marketing employees
* create reports about products (make queries).
* create special offers
* send special offers to inventory management

---
## 3. Inventory Management Module

### inventory employees
* add, delete, update, list, search (products)
* set minimum range for products

### system
* send notifiction
    - when products at special range (set by inventory employees)
    - when expiry date of product gets close.
* Manage the Damages items and sales Return.

---
## 4. Sales Module

### sellers (customers)
* search for product
* list all products
* make and cancel order

---
## 5. User Module

a) All Users can login and logout.
b) Users can Update their Information except ID.
c) Users can see all previous actions -optional-.
